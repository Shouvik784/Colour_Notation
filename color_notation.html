CSS Color Notations

A Comprehensive Guide to Color in Web Design
This interactive guide demonstrates various ways to specify and work with colors in CSS. Hover over the color boxes to see effects and click to copy the color code.

1. Named Colors

CSS provides 140+ predefined color names that can be used directly.

tomato
dodgerblue
mediumseagreen
slateblue
darkorange

2. HEX Colors

Hexadecimal notation using #RRGGBB format.

#e74c3c (Red)
#2ecc71 (Green)
#3498db (Blue)
#9b59b6 (Purple)
#f1c40f (Yellow)

3. RGB Colors

RGB notation using rgb(red, green, blue) where each value is between 0-255.

rgb(231, 76, 60)
rgb(46, 204, 113)
rgb(52, 152, 219)
rgb(155, 89, 182)
rgb(241, 196, 15)

4. RGBA Colors

RGB with Alpha channel for transparency (0 = fully transparent, 1 = fully opaque).

rgba(231, 76, 60, 0.8)
rgba(46, 204, 113, 0.6)
rgba(52, 152, 219, 0.4)
rgba(155, 89, 182, 0.8)
rgba(241, 196, 15, 0.6)

5. HSL Colors

Hue (0-360), Saturation (0%-100%), Lightness (0%-100%).

hsl(0, 100%, 50%)
hsl(120, 100%, 50%)
hsl(240, 100%, 50%)
hsl(60, 100%, 50%)
hsl(300, 100%, 50%)

6. HSLA Colors

HSL with Alpha channel for transparency (0 = fully transparent, 1 = fully opaque).

hsla(0, 100%, 50%, 0.7)
hsla(120, 100%, 50%, 0.7)
hsla(240, 100%, 50%, 0.7)
hsla(60, 100%, 50%, 0.7)
hsla(300, 100%, 50%, 0.7)

Color Combinations

This section shows how different color combinations affect readability and aesthetics.

High Contrast

Black on white or white on black provides maximum readability and accessibility for all users.

Complementary

Colors opposite each other on the color wheel create vibrant contrast and visual interest in your designs.

Analogous

Adjacent colors on the wheel create harmonious, naturally pleasing color schemes with subtle transitions.

Experiment with these color notations in your CSS to create beautiful designs!
